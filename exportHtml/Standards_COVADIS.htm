titreStandards COVADIS
commentReprésentation des modèles conceptuels de la COVADIS ; chaque standard est représenté par un vocabulaire et chaque classe géolocalisée est représentée par un concept dans ce vocabulaire. La structuration entre les classes est renforcée pour faciliter la compréhension, notamment quand une classe d'objets découle d'un autre classe. Les champs suivants des concepts sont utilisés: - le nom technique est reporté comme prefLabelFr - les synonymes sont reportés comme altLabelFr - la définition fournie par le standard est reportée comme definition - les informations complémentaires comme les regroupements ou les contraintes sont reportées comme scopeNote - les choix effectués pour cette transcription sont enregistrés dans editorialNote
